# 🌤️ Nimbus — Real-Time Weather Dashboard

Nimbus is a modern, premium weather dashboard built with **HTML, CSS, and Vanilla JavaScript**. It uses the **OpenWeatherMap API** to deliver real-time weather updates, hourly and 5-day forecasts, air quality information, and detailed wind insights through a sleek dark-themed interface.

🔗 **Live Demo:** https://sathvik983.github.io/Real-time-Weather-Dashboard/

---

## ✨ Features

* 🌤️ **Real-Time Weather** – Current temperature, feels-like temperature, humidity, pressure, visibility, wind speed, and sunrise/sunset.
* ⏱️ **24-Hour Forecast** – Scrollable hourly forecast with weather icons and rain probability.
* 📅 **5-Day Forecast** – Daily weather outlook with high/low temperatures and comparison bars.
* 🌬️ **Air Quality Index (AQI)** – Live AQI indicator with pollutant details including CO, NO₂, O₃, PM2.5, PM10, and SO₂.
* 🧭 **Interactive Wind Compass** – Displays wind direction and gust speed in real time.
* 🔍 **City Search with Autocomplete** – Quickly search for locations with instant suggestions.
* 📍 **Geolocation Support** – Fetch weather for your current location with one click.
* 🛠️ **Demo Mode** – Explore the dashboard using sample data without an API key.
* 🔄 **Unit Toggle** – Switch between Metric (°C, m/s) and Imperial (°F, mph).
* 💾 **Local Storage Caching** – Stores recent data for faster loading and a smoother experience.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Styling:** CSS Grid, Flexbox, Glassmorphism, Custom Animations
* **API Integration:** Fetch API, Async/Await, Promises
* **Storage:** Local Storage
* **Weather API:** OpenWeatherMap (Current Weather, Forecast, Geocoding, Air Pollution)

---

## 🚀 Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/sathvik983/Real-time-Weather-Dashboard.git
   ```

2. Navigate to the project folder.

3. Open `index.html` in your browser.

4. Choose one of the following:

   * **Demo Mode** – Explore the dashboard instantly without an API key.
   * **Live Mode** – Enter your personal OpenWeatherMap API key to view real-time weather data.
